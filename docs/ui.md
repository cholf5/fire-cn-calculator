# 五、关键交互说明

## 1）实时计算

* 任意输入变化 → 立即更新结果
* 无“计算按钮”

---

## 2）Preset 行为

点击按钮：

```
[节制] → 自动填充所有参数
[舒适] → 重置为默认
[轻奢] → 覆盖当前输入
```

---

## 3）滑杆 + 输入框联动

* 拖动滑杆 → 输入框同步变化
* 输入数字 → 滑杆位置更新

---

## 4）高级设置折叠

默认：

```
▼ 高级设置
```

点击展开：

```
▲ 高级设置
```

---

## 5）状态提示（视觉优先级）

| 元素     | 优先级 |
| ------ | --- |
| FIRE金额 | 最高  |
| 年支出    | 中   |
| 风险提示   | 低   |

# 七、UI 细节约束（避免 Codex 发散）

* 不要复杂动画
* 不要多页面（单页应用）
* 不要登录系统
* 不要图表库（初版用简单 div 实现）
